# Oskar Duş - Kurumsal Web Sitesi & Yönetim Paneli

Oskar Duş firması için geliştirilmiş, lüks ve modern bir tasarıma sahip kurumsal web sitesi projesidir. Müşterilerin firmanın tamamlanmış projelerini inceleyebileceği, sunulan hizmetler hakkında bilgi alabileceği ve online olarak "Ücretsiz Keşif ve Teklif" talebinde bulunabileceği tam kapsamlı bir platform olarak tasarlanmıştır.

## 🚀 Proje Özellikleri

*   **Modern ve Şık Kullanıcı Arayüzü (UI):** Firmanın marka kimliğine uygun, karanlık tema (dark mode) ağırlıklı şık tasarım.
*   **Dinamik Proje Sergileme:** Tamamlanan projelerin dinamik olarak listelendiği galeri ve portfolyo sayfaları.
*   **Teklif Alma Modülü:** Müşterilerin proje detaylarını girerek hızlıca fiyat teklifi ve keşif isteyebileceği iletişim ve form altyapısı.
*   **Hızlı İletişim (WhatsApp Entegrasyonu):** Kullanıcıların tek tıkla WhatsApp üzerinden iletişime geçebilmesini sağlayan modül.
*   **Yönetim Paneli (Admin Panel):** İçeriklerin, projelerin ve gelen teklif taleplerinin yönetilebildiği arka plan sistemi.
*   **Responsive Tasarım:** Mobil, tablet ve masaüstü cihazlarla tam uyumluluk.

## 💻 Kullanılan Teknolojiler (Geliştirici Notu: Burayı kendi kullandığın dillere göre güncelle)

*   **Frontend:** HTML5, CSS3, JavaScript (veya Angular/React)
*   **Backend:** .NET Core Web API / C#
*   **Veritabanı:** Microsoft SQL Server
*   **Mimari:** Clean Architecture & RESTful API prensipleri
